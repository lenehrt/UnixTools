The UnxUtils FAQ

  1. Bison complains:
            bison: /usr/local/share/bison.simple: No such file or directory
    Solution:
        You can set the location of bison.simple and bison.hairy by
                set BISON_SIMPLE=\\Machine\where\ever\is\bison.simple
            set BISON_HAIRY=\\Machine\where\ever\is\bison.hairy
